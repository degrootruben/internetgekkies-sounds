# Community Sound Packs

Want to create your own sound pack? See [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines.

## How to install a pack

```bash
mkdir -p ~/.claude/sound-packs
cd ~/.claude/sound-packs
git clone <pack-repo-url>
```

The plugin will automatically pick up any `.mp3` files in `<pack-name>/sounds/`.
